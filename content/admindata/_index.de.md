## Administrative Daten
Aufgrund der **Benutzerrechte** unterscheidet sich das statische Element **Administrative Daten** im **Navigationsbereich**. Das Element "**Benutzer Einstellung**" ist nur für einen Administrator sichtbar.

### Nachrichtensystem 
GT implementiert ein Nachrichtensystem, damit sich ein Benutzer an andere Benutzer bzw. Rolle wenden kann. Wobei dieses System primär für Nachrichten bezüglich Anliegen von **geteilten Daten** gedacht ist.
